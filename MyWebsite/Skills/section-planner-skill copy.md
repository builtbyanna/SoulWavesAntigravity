# Skill: section-planner

## Purpose
Plan a single website section before implementation.

## Inputs Required
- Section name
- Section goal
- Target audience
- Business context
- Visual direction

## Steps

### Step 1: Define purpose
Clarify what this section needs to achieve.

### Step 2: Define structure
Recommend heading, supporting copy, components, CTA, and visual elements.

### Step 3: Define motion
Suggest whether this section needs no motion, subtle reveal motion, or signature motion.

### Step 4: Define mobile behavior
Explain how the section should simplify on smaller screens.

### Step 5: Define implementation risks
Flag anything that could hurt clarity, performance, or SEO.

## Quality Checks
- [ ] Section purpose is clear
- [ ] Hierarchy is strong
- [ ] Motion is justified
- [ ] Mobile behavior is considered

## Output
A section blueprint ready for implementation.
