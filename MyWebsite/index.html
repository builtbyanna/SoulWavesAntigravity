About

I used to work in a prison.

Before all of this.

Before that: a kindergarten, a bakery, a horror escape theatre. None of those are on a typical freelancer's CV. But they gave me things that actually matter in this work: staying calm when things break, reading a situation fast, building structure where there isn't any, and seeing the human behind the process.

Before I left, I started messing around with HTML and CSS. I was not good at it. I still think code is mostly magic. I never got my Hogwarts letter, so this is as close as I get. I became a VA, chose to go into the tech direction, did some more coaching, and ended up building again.

I'm not a "traditional developer." I'm a builder who uses modern tools to solve the technical problems that keep you up at night. I'm honest about the process, I'm learning every day, and I don't stop until it's right.
